# IPL Match Predictor 🏏

This is a simple data-driven web application that predicts IPL match outcomes based on team, toss, and venue.

## Features

* Predicts match winner
* Uses venue-based performance data
* Displays Player of the Match
* Interactive UI using Streamlit

## Tech Stack

* Python
* Pandas
* Streamlit

## How to Run

1. Install dependencies:
   pip install streamlit pandas

2. Run the app:
   streamlit run app.py

## Dataset

Custom dataset created using basic team performance metrics.
